hexo博客搭建

2022-03-26 hexo 674 字
博客搭建
  • 搭建环境：Windows10
  • 没有安装git前，都是使用cmd输入命令

依赖工具安装

  • hondejs //cmd输入：node -v 验证
  • git //cmd输入：git --version 验证
    上面这个连个工具直接百度搜索下载安装，建议到腾讯软件搜索下载安装，更快捷。
  • hexo
    1、cmd输入：npm config set registry https://registry.npm.taobao.org //更改国内的npm镜像源下载hexo更快
    2、cmd输入：npm install -g hexo-cli //安装hexo
    输入：hexo -v //验证是否有版本号出现即为成功

初始化博客

  • cmd输入：hexo init blog //新建一个博客文件夹blog，并开始初始化博客（下载必要文件）
cd blog  //进入到blog文件下
npm install //下载安装
hexo s  //本地4000预览，这个命令需要在blog文件下执行
ctrl+c  //断开本地预览

hexo新建文章

hexo n 文章标题 //新建一篇文章

主题美化

  • 将主题下载或克隆到blog文件下的themes文件下
  • 修改blog文件夹下的_config.yml（称为：站配置文件）文件内的theme：主题名（主题名字就是克隆主题文件夹的名字）

博客部署

gitee

  • 注册gitee账号
  • cmd输入：shh-keygen //获取电脑公钥，公钥存放在C:\Users\用户名.ssh
  • 新建仓库（仓库名与用户名一致），例如：skillfans/skillfans

github

  • 注册账号
  • 新建仓库：仓库名为用户名+github.io

安装git部署工具

cd blog 
npm install hexo-deployer-git --save  //不安装就无法执行hexo d的命令无法部署到github/gitee

修改站配置_config.yml文件

站配置文件config.yml文件在blog文件夹下，主题配置文件config.yml在主题文件夹下

deploy:
type: git
repo: 仓库的SSH地址

部署

配置邮箱账户
回到桌面重新打开cmd命令窗口，或者使用右键打开【git bash here】,逐句输入以下命令回车

git config --global user. email "xxxxx@qq.com"  //配置邮箱

git config --global user.name "skillfans"  //配置用户名

hexo clean   //清理一下缓存

hexo s    //可以本地预览一下是否有错误   CTRL+c断开预览

hexo g   //生成静态文件

hexo d   //部署

部署到gitee仓库需要手动在仓库服务，点击Gitee Pase服务更新推送，首次使用gitee pase服务有一个实名认证，大概一天左右审核

部署到GitHub，秩序执行hexo d即可

博客更新

后续更新博客文章，执行以下命令即可。

hexo n 文章名字   //在blog文件的source问价中生成一篇博客.md文件

hexo clean  //清理缓存

hexo g  //生成静态文件

hexo d   //部署
