windows系统重装-官方工具

2020-03-29 windows 292 字
系统重装

下载官网系统安装工具

  • Windows10：https://www.microsoft.com/zh-cn/software-download/windows10
  • Windows11：https://www.microsoft.com/zh-cn/software-download/windows11

点击【下载工具】进行下载，Windows10同理

开始制作U盘

  • 选择【为另外一台电脑创建安装介质】

  • 取消【对这台电脑使用推荐选项】

  • 选择：语言、版本、体系结构

    • 版本：
      • Windows10：是一个综合版本：包括了（家庭版、专业版等）
      • Windows10家庭中文版：只有一个版本（家庭中文版）
    • 体系结构：
      • X64位
      • X86位（32位系统）

  • 使用介质选择U盘，当然后面需要镜像文件亦可以在这里下载

  • 等待下载安装介质完成即可

系统安装

  • 官网工具安装与U盘安装不同，不需要再在设置里关闭安全启动了，支持安全启动模式下安装
  • 插上U盘，连续点按F12进入快捷启动界面
  • 选择U盘
  • 跟着引导进行安装即可
  • 但要注意在磁盘分区哪里，注意资料备份数据无价！！！也可以删除所有分区，点击磁盘，点击应用即可自动分区
