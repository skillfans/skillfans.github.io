2020-03-29 windows 815 字
系统重装

在操作系统重装之前，请确保重要资料已经备份U盘或者除系统盘外的磁盘，电脑有价数据无价！

制作U盘启动

  • 分区格式

FAT32：不支持4G以上文件传输

NTFS：4G以上，稳定/可修复

exFAT：4G以上，U盘/SD卡/移动硬盘，跨平台win（XP不识别）/Mac/Linux/手机

  • 写入模式

USB-HDD：硬盘模式，兼容性高

USB-ZIP：软盘模式，兼容性不是很好

在制作PE工具U盘中，建议写入模式：USB-HDD 分区格式：NTFS

BIOS设置

  • 安全启动（secure boot）

现在的电脑一般都是UEFI引导模式了，很多时候我们装不了系统就是因为我们没有关闭这个安全启动导致的，所以建议在重装Windows系统时建议先进入BIOS设置界面关闭安全启动，特别是在降级安装系统的时候更需要关闭安全启动了。

进入BIOS的快捷键，台式电脑大部分是开机连续点按F2，笔记本也是如此，如无效可百度该品牌电脑进入BIOS设置的快捷键。


设置完成后就需要进入到PE系统中了，在重启过程中连续点按F12（大部分台式机笔记本，有异百度一下品牌），选择U盘名称回车键确定，

磁盘分区

如果没有重要文件，可以选择分区重装系统，如果有重要文件，建议只格式化C盘，数据无价！

分区对齐

2048机械

4096固态

ESP分区：UEFI引导文件

MSR分区: Microsoft 保留 (MSR) 分区

  • 分区表类：

MBR：针对Windows7，老系统传统bios引导，建议选择

GUID：Windows10，UEFI引导的系统，最近新出的建议选择

系统安装

  • 一键安装器

选择系统镜像文件，选择安装磁盘，一键安装即可。

  • windows安装器

选择系统镜像文件，安装位置，驱动目录与安装位置相同，选择系统版本，安装即可。

需要注意的是，右边三个指示灯不能为红，如果遇到引导驱动器和安装位置都选择C盘，指示灯变红，可以通过引导修复 工具解决这个问题。

  • ghost安装

ghost安装一次点击，开始选择系统镜像文件，安装位置，点击ok，yes即可进行安装。
